# RDP STUDIO - Hub Portfólio Profissional

Este repositório contém uma coleção de ferramentas e projetos de engenharia de software desenvolvidos por **Marcelo Rodrigues** sob a marca **RDP STUDIO**. O objetivo é demonstrar competência técnica, design de interface e resolução de problemas reais.

## Projetos em Destaque

### 1. Validador de MACs (Segurança)
**Stack:** JavaScript, RegEx, DOM Manipulation
Ferramenta para sanitização de listas de endereços MAC e geração automática de scripts de configuração para Firewalls Fortigate.

### 2. Automação GLPI (Automação)
**Stack:** Python (Lógica), Web Interface
Agente inteligente para abertura massiva de chamados, integrando-se à API do GLPI para upload de evidências e roteamento.

### 3. Epic Games Scanner (Geek/IA)
**Stack:** React, Tailwind CSS
Sistema de monitoramento em tempo real para detecção de ofertas e promoções, com interface "hacker/terminal".
